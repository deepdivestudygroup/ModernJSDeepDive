## 스터디 최종 목표

- 모던 자바 스크립트 딥 다이브 책 완독

- 꾸준히 공부하는 습관 기르기

## 스터디 방식

- 6일간 스터디 준비
  1. 책 읽기
  2. Github에 정리하기, 정리 방식은 자유 (해당 주차 폴더에 "본인이름.md" 파일에다가 정리하기!)
  3. 해당 주차에서 공부한 내용을 토대로 퀴즈 준비하기
 
- 스터디 당일
  1. 정리한지 확인
  2. 퀴즈 풀기
  3. 의견이 갈리는 내용들은 토론하기
  4. 공부하며 인상 깊은 것들 공유하기
  5. 다음 주차 어떻게 공부할지 살짝 공유 (예고)
